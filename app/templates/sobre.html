Sobre o Projeto

Este projeto tem como objetivo desenvolver um sistema de detecção de fogo e fumaça utilizando visão computacional. O foco é a criação de uma solução eficiente e em tempo real para identificar focos de incêndio e minimizar os riscos em ambientes vulneráveis.

Objetivos

  • Desenvolver uma ferramenta que utiliza inteligência artificial para detectar fogo e fumaça.
  • Integrar o sistema com diferentes dispositivos, como câmeras de segurança, webcams e vídeos de arquivos.
  • Utilizar modelos de aprendizado de máquina, como CNNs, para realizar a detecção de forma eficiente.
  • Proporcionar uma interface amigável para o usuário, permitindo o uso tanto via upload de vídeos quanto em tempo real através de uma webcam.

Tecnologias Utilizadas

  • Flask: Framework web para backend.
  • TensorFlow e Keras: Utilizados para treinar e carregar os modelos de aprendizado profundo.
  • OpenCV: Biblioteca para manipulação de imagens e vídeos.
  • Bootstrap: Framework CSS para estilização e criação de uma interface responsiva.

Equipe de Desenvolvimento

O projeto foi desenvolvido pelos alunos da turma EC9 como parte do trabalho de conclusão de curso, com foco na implementação prática de técnicas de visão computacional em cenários de monitoramento de incêndios.

Motivação

A motivação para este projeto surge da crescente demanda por sistemas de monitoramento mais eficientes em áreas florestais, indústrias e até residências. Com a utilização de visão computacional, é possível detectar de forma rápida e precisa focos de incêndio, evitando a propagação de danos e salvando vidas e propriedades.
